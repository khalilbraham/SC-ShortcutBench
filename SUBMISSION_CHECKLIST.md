# NeurIPS 2026 Submission Checklist

Complete checklist for submitting SC-ShortcutBench paper to NeurIPS Evaluation & Datasets track.

## Manuscript

- [ ] **Main paper** (`paper.tex`, compiled to `paper.pdf`)
  - [ ] All sections completed (Introduction through Conclusion)
  - [ ] All tables integrated (Tables 1-9)
  - [ ] All figures included
  - [ ] References complete and formatted
  - [ ] Page limit met (typically 8 pages + 4 pages appendix for NeurIPS)

- [ ] **Supplementary material** (if submitted)
  - [ ] Full appendix with all proofs
  - [ ] Additional tables and figures
  - [ ] Extended analyses
  - [ ] All ~15 appendix sections from draft

## Code & Scripts

Reproducible pipeline for NeurIPS Datasets submission:

- [ ] **Pipeline scripts** (`scripts/`)
  - [ ] `run_benchmark.py` - Main orchestration (Stage 1-6)
  - [ ] `run_analysis.py` - Post-hoc analysis & figures
  - [ ] `data_loading.py` - CELLxGENE Census loading
  - [ ] `conflict_rows.py` - Algorithm 1 implementation
  - [ ] `evaluate_encoders.py` - Representation + prediction audit
  - [ ] `evaluate_generation.py` - Generative model evaluation
  - [ ] `evaluate_baselines.py` - Baseline comparisons
  - [ ] `analysis.py` - Statistical analysis
  - [ ] `utils/statistical.py` - Grouped bootstrap CIs
  - [ ] `utils/metrics.py` - Evaluation metrics
  - [ ] `README.md` - Usage guide

- [ ] **Configuration files** (`configs/`)
  - [ ] `benchmark_config.yaml` - Pipeline configuration
  - [ ] `analysis_config.yaml` - Analysis settings
  - [ ] `models.yaml` - Model definitions

- [ ] **Dependencies**
  - [ ] `requirements.txt` - Complete Python dependencies
  - [ ] Installation instructions in README
  - [ ] Tested on Python 3.8+

## Benchmark Validation

From paper methodology:

- [ ] **Corpus evidence** (Section 2.1)
  - [ ] Metadata lookups solve large fraction (Table 2)
  - [ ] Source identity recoverable from raw expression (Table A2)
  - [ ] Multi-route shortcut structure (Appendix A.1)
  - [ ] Shortcut survival under source-holdout (Appendix A.2)

- [ ] **Encoder audit** (Section 3.1)
  - [ ] Linear recoverability ρ(Z; φ) computed (Table 3)
  - [ ] SDR > 1 for all encoders
  - [ ] All 5 encoders evaluated

- [ ] **Prediction audit** (Section 3.2)
  - [ ] TSM computed for all task-encoder pairs
  - [ ] Decorrelated tissue TSM: -24.7 to -31.1 pp
  - [ ] Direct vs downstream probe comparison (Table 6)
  - [ ] Both-seen label support control

- [ ] **Generation audit** (Section 3.3)
  - [ ] Native output evaluation (Table 8)
  - [ ] Generative interfaces tested (5 models)
  - [ ] Prompt intervention (Table 9)
  - [ ] Expression-only, shortcut-context, anti-shortcut conditions

- [ ] **Statistical analysis**
  - [ ] 95% grouped-bootstrap CIs with grouping at dataset_id
  - [ ] Dose-response with prior purity (Table 7)
  - [ ] Transition maps (Table 6 supplement)
  - [ ] Embedding geometry analysis

## Paper-Specific Sections

- [ ] **Introduction**
  - [ ] Problem motivation clear
  - [ ] Related work properly cited
  - [ ] Contribution statement explicit

- [ ] **Methods** (Section 2-3)
  - [ ] Algorithm 1 pseudocode
  - [ ] Task definitions
  - [ ] Evaluation protocols for each access mode

- [ ] **Results** (Section 3)
  - [ ] Corpus evidence compelling (Tables 2, A1, A2)
  - [ ] Encoder audit results complete (Tables 3-7)
  - [ ] Generation results comprehensive (Tables 8-9)
  - [ ] Mechanism analyses complete (Section 4)

- [ ] **Theory** (Section 3.5)
  - [ ] Structural causal model figure and explanation
  - [ ] Proposition 1 statement and proof sketch
  - [ ] Load-bearing assumptions discussed

- [ ] **Mechanism** (Section 4)
  - [ ] Geometric analysis (Figure 2, Table 7)
  - [ ] Transition maps (Table 6)
  - [ ] Dose-response (Figure 3)
  - [ ] Prompt intervention (Table 9)

- [ ] **Recommendations** (Section 5)
  - [ ] Four operational recommendations
  - [ ] Each tied to specific result
  - [ ] Implementable without new methods

- [ ] **Limitations** (Section 6)
  - [ ] Conflict rows are constructed, not natural
  - [ ] Proposition 1 is sufficient, not necessary
  - [ ] Only frozen encoders evaluated
  - [ ] Disease effects weaker than tissue

  - [ ] README with instructions


  - [ ] File size under limit (typically 100MB)
